Introduction

Ce mini-site a été crée dans le cadre du cours de TP Programmation web.

Il permet de découvrir la création de plusieurs pages HTML reliees entre elles.

Objectifs du mini-site

L' objectif est d'apprendre à apprendre à structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page Accueil présente le contexte général du mini-site.
  • La page Presentation contient les Informations personnelles de l'etudiant.
  • La page contact contient les Informations de conctactde l'etudiant

Je veux apprendre la programmation web afin de creer des outils utiles pour la supervision, l'administration des reseaux et la securisation des systemes informatiques
