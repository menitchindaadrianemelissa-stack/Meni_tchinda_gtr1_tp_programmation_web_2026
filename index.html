Introduction

Ce mini-site a été créé dans le cadre du cours de tp programmation web

Il permet de découvrir la création de plusieurs pages HTML reliées entre elles.

Objectif du mini-site

L'objectif est d'apprendre a structurer un petite project web avec plusieurs fichiers.

Pages disponibles

  • La page Acceuil présente le contexte général du mini-site.
  • La page présentation contient les informations personnelles de l'étudiant.
  • La page contact contient les informations de contact de l'étudiant.

J'apprends la programmatrion web afin de creer des sites web et applications, développer des compétences numériques et acceder ainsi a de meilleurs opportunités profesionnelles bien rémunérées.
